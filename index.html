sphereCrypt

sphereCrypt is a cryptography algorithm which I created, using a custom encryption / decryption technique. The algorithm allows string messages to be encrypted to a hex string, which can then be passed into the decryption scheme which will return the original message. The encryption is not one-to-one, meaning that the same message may be encrypted in many different ways. Despite this, it can always be decrypted back into the original message.
